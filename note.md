# Important_HotKeys

- Open link (Ctrl + link)
- Toggle edit/preview (Ctrl + e)
- Open quick switchers (Ctrl + o)
- Search (Ctrl + shift + f)
- Find (Ctrl + f)
- Back (Ctrl + Alt + leftarrow)
- Forward (Ctrl + Alt + Rightarrow)
- New note (Ctrl + n)
- Open note in new window (Ctrl + click on note)
